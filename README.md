Lihat website live di: ajiprasetia.vercel.app

Modern UI/UX: Desain yang bersih dan responsif

Responsive Design: Tampilan yang optimal di semua perangkat
SEO Friendly: Struktur yang baik untuk mesin pencari

📋 Prerequisites
Sebelum memulai, pastikan Anda sudah menginstall:

Node.js (versi 16 atau lebih tinggi)
npm atau yarn
🔧 Instalasi
Clone repository ini
git clone https://github.com/ajiprasetia/my-portfolio.git
Masuk ke direktori project
cd my-portfolio
Install dependencies
npm install
atau jika menggunakan yarn:

yarn install
🚀 Menjalankan Project
Development Mode
Untuk menjalankan project dalam mode development:

npm run dev
atau:

yarn dev
Aplikasi akan berjalan di http://localhost:5173 (atau port lain jika 5173 sudah digunakan).

Build untuk Production
Untuk membuat build production:

npm run build
atau:

yarn build
File hasil build akan tersimpan di folder dist/.

Preview Build
Untuk melihat preview dari build production:

npm run preview
atau:

yarn preview
🎨 Customization
Mengubah Konten
Edit komponen di folder src/components/ untuk mengubah konten portfolio
Ganti gambar di folder src/assets/ atau public/
Mengubah Styling
Konfigurasi Tailwind CSS dapat diubah di tailwind.config.js
Custom CSS dapat ditambahkan di file CSS yang ada atau membuat file baru
📦 Scripts yang Tersedia
Script	Deskripsi
npm run dev	Menjalankan development server
npm run build	Membuat production build
npm run preview	Preview production build
npm run lint	Menjalankan ESLint untuk check code quality
🚢 Deployment
Project ini sudah dikonfigurasi untuk deployment di Vercel. Untuk deploy:

Push code ke repository GitHub
Import project di Vercel
Vercel akan otomatis mendeteksi Vite dan melakukan build
Website akan live dalam beberapa menit
Alternatif deployment:

Netlify: Drag & drop folder dist/ atau connect ke GitHub
GitHub Pages: Gunakan GitHub Actions untuk auto-deploy
Railway: Connect repository dan deploy
